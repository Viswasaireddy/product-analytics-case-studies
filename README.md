# Product Analytics Case Studies

This repository contains product analytics and data analysis projects focusing on user behavior, growth analytics, experimentation, and unit economics.

## Projects

### 1. Multicity Unit Economics & Margin Optimization
Analyzed 100k+ marketplace transactions to diagnose profitability issues across cities and identify cost drivers.

Key results:
- Improved contribution margin by 14%
- Reduced variable costs by 9%
- Identified ₹22L annual revenue opportunity

Project file:
unit-economics-analysis.md

---

### 2. Growth & Retention Analysis
Analyzed 50k+ user dataset to identify churn patterns and onboarding friction.

Key results:
- Identified 32% first-month churn
- Proposed strategies improving retention by 11%
- Reduced churn by 8%

Project file:
growth-retention-analysis.md

---

### 3. Cohort Retention & Repeat Purchase Analysis
Analyzed repeat purchase patterns and cohort retention behavior across customer segments.

Key results:
- Increased repeat purchase rate by 10%
- Improved customer lifetime value by 15%

Project file:
cohort-retention-analysis.md

---

### 4. A/B Experimentation & Incentive Optimization
Evaluated new incentive feature through A/B testing on 20k+ users.

Key results:
- 4.6% statistically significant conversion uplift
- Projected 6–8% MRR increase

Project file:
ab-testing-analysis.md
