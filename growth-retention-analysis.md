# Growth & Retention Analysis

## Problem
Analyzed a dataset of 50k+ users and identified a 32% first-month churn rate indicating friction in the onboarding journey.

## Action
- Performed SQL cohort analysis using CTEs and window functions
- Tracked acquisition and retention trends across monthly cohorts
- Identified onboarding drop-offs and user lifecycle behavior

## Impact
- Proposed strategies to improve 30-day retention by 11%
- Reduce churn by 8%
- Potentially save ₹5L+ in monthly acquisition spend

## Tools Used
SQL, Cohort Analysis, Data Analysis, Product Analytics
