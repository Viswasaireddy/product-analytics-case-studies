# Cohort Retention & Repeat Purchase Analysis

## Problem
Identified inconsistent repeat purchase behavior across 12+ customer segments, limiting long-term customer lifetime value (LTV).

## Action
- Built SQL-based cohort retention tables using joins and aggregations
- Analyzed customer acquisition cohorts and repeat purchase patterns
- Examined the first 14-day lifecycle window to detect early churn signals

## Impact
- Designed re-engagement strategies projected to increase repeat purchase rate by 10%
- Improved long-term customer lifetime value (LTV) by approximately 15%

## Tools Used
SQL, Cohort Analysis, Customer Retention Analysis, Data Analysis
